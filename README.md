- 包括什么
```c
一种语法,是为了写表达式
一个正则表达式 是一个 字符串,
	字符串里面的全部字符(物理上)包括
		普通字符 
                    原义字符
		元字符(这个说的是扩展正则语法) 
                    ^ $ . [  ] {  } - ? * + (  ) | \
	而逻辑上包括
		一个字符集 
		锚
		修饰符
	
而 正则语法 分为
	标准
          ^ $ . [  ] *
	扩展(比 标准 多几个元字符)
          (  ) {  } ? + |

不同工具以及不同语言之间的正则表达式都略有差异。
```
